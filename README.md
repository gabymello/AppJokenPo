App Jokenpô Mobile
📱 Sobre o Projeto

O App Jokenpô Mobile é um aplicativo desenvolvido no Android Studio utilizando Java/XML, inspirado no clássico jogo de Pedra, Papel e Tesoura.

O objetivo do projeto é permitir que o usuário escolha uma das opções disponíveis enquanto o aplicativo gera uma jogada aleatória, exibindo o resultado da partida na tela.

🎮 Funcionalidades
Escolha entre:
Pedra
Papel
Tesoura
Geração automática da escolha do aplicativo
Exibição da imagem escolhida pelo aplicativo
Resultado da rodada:
Vitória
Derrota
Empate
Interface simples e intuitiva

Android Studio

🚀 Como Criar o Projeto
1. Preparação

Antes de começar, é necessário possuir o Android Studio instalado no computador.

Caso ainda não tenha:

Faça o download do Android Studio
Instale normalmente
2. Criando um Novo Projeto
Abra o Android Studio
Clique em:
New Project
Escolha:
Empty Activity
Nomeie o projeto como:
AppJokenpo
Clique em:
Finish
🎨 Estrutura do Layout

No arquivo:

activity_main.xml

foi desenvolvido um layout contendo:

3 botões:
Pedra
Papel
Tesoura
1 ImageView:
Responsável por mostrar a escolha do aplicativo
1 TextView:
Responsável por exibir o resultado da partida
🧠 Funcionamento da Lógica

O aplicativo funciona da seguinte forma:

O usuário seleciona uma opção
O sistema gera uma escolha aleatória
As escolhas são comparadas
O resultado é exibido na tela
